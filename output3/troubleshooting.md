# Troubleshooting Guide

## Common Issues
- **Python not found**: Ensure Python 3.7+ is installed and in PATH.
- **Incorrect output**: Double-check input values and customer type string (case-insensitive).
- **Syntax error**: Verify file integrity and Python version compatibility.

## Diagnostic Steps
1. Run `python --version` to confirm Python installation.
2. Run `python discount_calculator.py` and check for expected outputs.
3. Review code for modifications or accidental changes.

## Support
- For issues, open a GitHub issue in the repository.
- Reference original Java code for logic comparison.
