# Troubleshooting Guide: DiscountCalculator Python Migration

## Common Issues
- **Syntax Errors**: Ensure Python 3.8+ is installed and used
- **Logic Mismatches**: Compare outputs with Java implementation; review conversion notes
- **Test Failures**: Update or add test cases for new business logic

## Diagnostic Steps
1. Review `conversion_errors.json` for logged errors
2. Run `discount_calculator.py` with sample inputs; verify outputs
3. Check for typos or missing dependencies

## Support Resources
- Contact modernization team for migration or integration support
- Refer to `migration_guide.md` for setup and usage
- Consult `validation_report.md` for conversion details

## Advanced Troubleshooting
- Enable verbose logging in Python for detailed error output
- Use Python linters (`flake8`, `pylint`) for code quality checks

## Escalation
- For unresolved issues, document steps taken and notify the modernization support team
